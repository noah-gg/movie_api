myFlix

A movie web application

This is a REST API for an application called “myFlix”. It allows the users access to information on different movies, genres and directors. Also allows users to sign up / deregister, update user information and create / modify a list of their favourite movies.

Description URL Endpoint HTTP Method Query parameters Request body data format Response body data format
Return a list of all movies to the user /movies GET None None A JSON object holding all data about all the movies
Return data about single movie by title to user /movies/[title] GET :title None A JSON object holding all data about a selected movie 

{ 
	title:"Bubble", 
	description:" ",
	director:"Tetsuro Araki", 
	genre:"Japanese" "Sci-fi Films" "Romance Anime",
}
Return data about genre /movies/genre/[genreName] GET :genre None A JSON object holding all data about a selected genre 

{ 
	genre:"Sci-fi Films",
	description: " ",
}
Return data about director movies/director/[directorName] GET :director None A JSON object holding all data about a selected director 

{ 
	director:"Tetsuro Araki",
	bio: " ",
}
Allow new users to register /users POST None A JSON object holding data about the user to add, structured like: 

{
	id: 1,
	userName: " ",
	fullName: " ",
	emailAddress: " ",
	favourites: [],
}
A JSON object holding data about the user that was added, structured like: 

{
	id: 1,
	userName: " ",	
	fullName: " ",
	emailAddress: " ",
	favourites: [],
}
Allow users to update their user info /users/[userName] PUT :userName Data containing old user information A text message indicating the user information was succesfully updated.
Allows existing user to deregister /users/[userName]/delete DELETE :userName None A text message indicating the user the account has been deleted.
Allow users to add movie to favourites /users/[userName]/favourites POST :userName :favourites None A text message indicating movie was succesfully added to favourites.
Allow users to delete movie from favourites /users/[userName]/favourites DELETE :userName :favourites None A text message indicating movie is removed from favourites.
